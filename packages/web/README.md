# @web-component-analyzer/web

Web application for visualizing component DFDs in the browser.

## Features

- Paste component code directly
- Real-time DFD visualization
- Interactive Mermaid diagrams
- No installation required

## Development

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
```

Output will be in `dist/` directory, ready for deployment to GitHub Pages.
